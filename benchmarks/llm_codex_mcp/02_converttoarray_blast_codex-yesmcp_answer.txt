**If Return Type Changes To Object**

- If you simply flip `json_decode(..., true)` to object mode, the helper itself is likely to break before callers do. Its sanitization loop in [app/helpers.php#L1310](/Users/naman/Documents/QBApps/qbapi/app/helpers.php#L1310) treats non-arrays as scalar values and passes them into `htmlentities`; `stdClass` values would fail there.
